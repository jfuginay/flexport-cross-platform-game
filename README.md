## 🚀 **Getting Started**

### **iOS**
```bash
cd "FlexPort iOS"
open FlexPort.xcodeproj
# Build and run on simulator or device
```

### **Android**
```bash
cd "FlexPort Android"
./gradlew assembleDebug
adb install app/build/outputs/apk/debug/app-debug.apk
```

### **Web**
```bash
cd Web
npm install
npm run dev
```
